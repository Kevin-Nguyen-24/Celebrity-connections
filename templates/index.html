How it works

This tool uses the Wikipedia API to find the shortest path between two celebrities through Wikipedia article links. It employs a breadth-first search (BFS) algorithm to discover connections.

What is a connection? Two celebrities are connected when one person's Wikipedia page links to another's. The tool finds the shortest chain of these Wikipedia links between any two people.
